# 🧭 SummitCare Developer Guide — Local Dev & Verification

## 🚀 Core Commands

### Command Purpose

```bash
pnpm run dev	        # Start your local app (frontend via Vite or Next.js, depending on setup)
pnpm run vercel:dev	    # Run your app using Vercel’s local environment — simulates production
pnpm run vercel:build	#Build locally with the same process Vercel uses for deployments
```
